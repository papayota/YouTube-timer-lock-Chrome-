# YouTube Timer Lock

YouTube へのアクセスをタイマー式でロックし、作業に集中するための Chrome 拡張機能（Manifest V3）です。

## 機能

- **タイマー式ロック**: 30分 / 60分 / 90分 / 120分 から選択してYouTubeをロック
- **完全ブロック**: ロック中はYouTube（Shorts含む）を一切開けません
- **ブラウザ再起動後も継続**: ロック状態は `chrome.storage.local` に保存され、ブラウザを閉じても継続
- **自動解除**: 設定時間が経過すると自動的にロック解除
- **手動解除**: ポップアップから2段階確認で解除可能
- **ブロック画面**: ロック中にYouTubeへアクセスすると、残り時間とユーモアのあるメッセージを表示

## インストール方法

1. このフォルダ全体をダウンロードまたはクローン
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダ（`youtube-timer-lock`）を選択

## 使い方

### ロック開始
1. 拡張機能アイコンをクリックしてポップアップを開く
2. 30分 / 60分 / 90分 / 120分 から希望の時間を選択
3. 「ロック開始」ボタンをクリック

### ロック中
- YouTubeへアクセスしようとすると、ブロック画面が表示されます
- ブロック画面には残り時間とメッセージが表示されます
- 「作業に戻る」ボタンでタブを閉じて作業に戻れます

### ロック解除
1. 拡張機能アイコンをクリック
2. 「解除する…」ボタンをクリック
3. 確認画面で「解除する（実行）」をクリック

## 技術仕様

- **Manifest Version**: 3
- **ブロック方式**: `chrome.declarativeNetRequest` (DNR)
- **状態管理**: `chrome.storage.local`
- **自動解除**: `chrome.alarms`
- **対象ドメイン**: `*.youtube.com/*`, `youtu.be/*`

## 注意事項

- 外部APIは使用していません（完全ローカル動作）
- ロック中でもポップアップから解除できます（2段階確認）
- ブラウザを閉じてもロック状態は継続します
